# Shimless RMA
## What is it?
Shimless RMA is google's way of recovering without a RMA Shim.
## How can I use it?
* Hold power
* tap refresh 3 times
* release power
## What can it do?
* Disable Write Protection
* Install custom firmware
* And more!
